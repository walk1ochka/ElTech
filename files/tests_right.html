ТЕСТИРОВАНИЕ

1. Для проведения тестирования на ПК необходимо инсталлировать с компакт-диска (или с флешки) на Ваш компьютер тестовую систему UTIS. Инсталляция пройдет автоматически на диск "С" вашего компьютера после щелка мышью на выделенном слове УСТАНОВИТЬ. Для проведения тестирования в локальной сети дисплейного класса тестовую ситему UTIS нужно установить с компакт-диска (или с флешки) на сервер сети.

2. Создать свою базу тестовых заданий или увеличить существующую можно в редакторе тестов UTISB, описание которого приведено в соответствующем файле дерева, расположенного в левом поле (UTISB).

3. Тестируемому по математике, физике, электротехнике (на первом уровне) по темам дисциплины и по группам тем последовательно выводятся на экран дисплея 20...24 тестовые задания, которые выбираются случайным образом из баз, в которых соответственно 80, 80 и 500 тестовых заданий. Время тестирования не ограничено. При желании можно установить (воспользовавшись программой UTISB/Проект) другие параметры тестирования (число заданий, время сессии тестирования, форму вывода результатов тестирования и др.).
4. В качестве примера организации письменного итогового тестировании по дисциплинам: "Электротехника" и "Основы теории сигналов" на втором уровне, приведена программа выбора (случайным образом) двух заданий из базы, содержащей 100 заданий 2-го уровня, или одного задания из базы, содержащей 16 аккредитационных заданий, подготовленных для предъявления студентам при проведении аккредитации (лицензировании) кафедры (см. п. 3. и п. 4 в поле слева)
