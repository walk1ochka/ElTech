1. ПРОГРАММНЫЙ  УЧЕБНО-МЕТОДИЧЕСКИЙ КОМПЛЕКС  ПО ЭЛЕКТРОТЕХНИКЕ

 (МОДУЛЬ УМК-Э1 базовой  дисциплины для вузов «ЭЛЕКТРОТЕХНИКА И ЭЛЕКТРОНИКА»)

Программный учебно-методический комплекс по электротехнике УМК-Э1 (UMK-E1) предназначен для очного, очно-заочного и дистанционного обучения студентов вузов, обучающихся по неэлектротехническим направлениям подготовки бакалавров и неэлектротехническим направлениям подготовки специалистов.

В комплекс УМК-Э1 включены: примерная программа базовой дисциплины для вузов «Электротехника и электроника», основные ГОСТы по электротехнике, электронный курс модуля «Электротехника», соответствующий учебнику "Марченко А.Л., Опадчий Ю.Ф. Электротехника и электроника. В 2-х т. Т. 1: Электротехника. Учебник. - М.: ИНФРА-М. 2018. - 274 с.", виртуальный лабораторный практикум по электротехнике, курсовые работы, упражнения, задания, тесты для самоконтроля, интерактивные тренажеры, глоссарий, система озвучивания, электротехнический калькулятор ElCalc, аннотации авторских электронных ресурсов (рис. 1).

Рис. 1. Компоненты УМК по электротехнике

В основу разработанного комплекса УМК-Э1 положена технология модульного обучения. В нём фрагменты основ теории соответствующей темы сопровождаются интерактивными упражнениями и заданиями с программной проверкой правильности расчётов, лабораторными работами на моделях и курсовыми работами; тестовыми заданиями ко всем темам модуля «Электротехника».

Комплекс УМК-Э1 разработан с использованием:

· языка разметки с добавлением каскадных стилей и Java–скриптов (оболочка, электронный курс, глоссарий, программа дисциплины и др.);
· среды Borland C++ Builder (программы LabWorks, ElCalc, RLCQuadro, Resonance, LogH, DF, Magn_C, Lr19, Lr20 и др.);
· среды Macromedia Flash MX, Adobe Flash 9 и выше (интерактивные упражнения, задания, тесты и др.);
· среды схемотехнического моделирования NI Multisim (лаб. практикум);
· языка Delphi (тестовая система UTIS, программа Kurs 2);
· редакторов Macromedia Dreamweaver MX, MS Word, Adobe Photoshop, Adobe Acrobat.
Тип ЭВМ: IBM PC, CD-ROM, 1240 x 1024 (1240 х 800 для ноутбуков).
Тип и версия ОС: IBM; для УМК-Э1 – ОС не ниже Windows 7, оперативной памяти не менее 512 МВ, 1,5 ГВ дискового пространства, браузер не ниже MSIE9.
