Задание 5.2
Определить магнитные потоки в стержнях магнитопровода магнитной цепи (рис. 5.26а), для которой заданы l1м = l2м, l3м, δ, S1 = S2 ≈ Sδ, S3 = 2S2; МДС F1 = w1I1, F2 = w2I2, F3 = w3I3 (F1 = F2 > F3) обмоток и магнитная характеристика В(Н) стали 3411 (см. рис. 5.17б) магнитопровода.
Указание. Расчёт можно выполнить с помощью учебной программы Мagn_C (для её запуска щёлкните мышью на имени файла и следуйте предлагаемой последовательности действий). Значения параметров магнитной цепи может быть сформировано случайным образом или введено вручную.
Примечание. Светный фон в программе проявляется при работе на ПК с браузером MSIE6 и ОС ХР/2000 Windows (см. Авторские эл. ресурсы, п. 8).
