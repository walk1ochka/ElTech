Задание 6.2

Определить магнитные потоки в стержнях магнитопровода магнитной цепи (рис. 6.26, а), для которой заданы l1М = l2М, l3М, d; S1 = S2 » Sd, S3 = 2S2; МДС F1 = w1I1, F2 = w2I2, F3 = w3I3 (F1 = F2 > F3) обмоток и магнитная характеристика В(Н) стали 3411 (см. рис. 6.17, б) магнитопровода.


Указание. Расчёт можно выполнить с помощью учебной программы Мagn_C (для её запуска щёлкните мышью на имени файла и следуйте предлагаемой последовательности действий). Значения параметров магнитной цепи может быть сформировано случайным образом или введено вручную.
