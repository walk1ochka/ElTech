Укажите, с какой целью на щиток двигателя выводятся начала и концы секций обмоток статора?


Для подключения обмоток статора к трехфазной сети переменного тока.

Для удобства замены катушек статорной обмотки при ремонте

Для замены местами двух проводов из трех, подводящих трехфазное напряжение сети к началам обмоток статора, для реверсирования АД.

Для изменения числа пар полюсов магнитного поля статора и изменения синхронной скорости АД.
