3. МОДЕЛИРОВАНИЕ СХЕМ ИССЛЕДУЕМЫХ ЦЕПЕЙ

Для исследования динамических свойств цепей (звеньев) нужно собрать на рабочем поле программы MS10 обобщенную схему цепи (рис. 17.3) (или открыть файл 17.3.ms10).

С помощью переключателей, управляемых клавишами S, D и F, можно замкнуть накоротко соответственно элементы R1, L1 и C1 первой ветви, а с помощью переключателей, управляемых клавишами Q, W и T, можно отключить (или включить в схему) элементы R2, L2 и C2.

Схема подключается к источнику постоянного напряжения с ЭДС Е посредством управляемого временем контакта Р. Для моделирования вход­ного воздействия 1(t) и получения реакции (переходной функции hu(t) цепи по напряжению) установим следующие параметры управляемого контакта Р: Time  on = 11 ms (11 мс); Time off = 1 ms (1 мкс). После запуска программы MS10 цепь подключается к источнику Е = 1 В через интервал Time off = 1 мкс на время t » 11 мс, достаточное для практического завершения свободных колебаний в цепи.

Для моделирования входного воздействия d(t) и получения реакции цепи (им­пульсной функции gu(t) по напряжению) установим ЭДС источника  Е  = 1 кВ; Time  on = 11 ms (11 мкс); Time off = 1 ms (1 мкс) переключателя Р. После запуска программы MS10 цепь подключается к источнику напряжения с ЭДС Е через интервал Time off  = 1 мкс на время tи = 10 мкс.
