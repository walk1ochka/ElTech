УПРАЖНЕНИЕ 8.1

Трехфазный асинхронный двигатель с короткозамкнутым ротором имеет номинальные данные: мощность по валу Р2н = 10 кВт; линейное напряжение Uн = 380 В при ƒ = 50 Гц; схема соединения обмоток статора - Y; число пар полюсов p = 2; частота вращения ротора n2н = 1450 об/мин; КПД ηн = 87,5%; коэффициент мощности cosφн = 0,88; кратность пускового тока α = Iп/Iн = 5; кратность пускового момента β = Mп/Mн = 1,2; кратность максимального момента λ = Mmax/Mн = 2. Определить мощность и ток, потребляемые двигателем из сети при номинальной нагрузке; пусковой ток; как изменится пусковой момент двигателя при снижении напряжения на его зажимах на 10% и возможен ли пуск двигателя с номинальной нагрузкой при изменившихся условиях. Построить естественную механическую характеристику n2 = ƒ(M) при изменении скольжения в пределах 0 ≤ S ≤ 1.

Решение.
5. Для построения механической характеристики определим частоту вращения магнитного поля статора
     
и номинальное скольжение
     
Принимая значения скольжения S = (0,25; 0,5; 0,75; 1)Sn, вычислим соответствующие им частоты вращения ротора
     
и вращающие моменты по формуле Клосса
     
где - критическое скольжение ротора, при котором вращающий момент М = Мmax.
