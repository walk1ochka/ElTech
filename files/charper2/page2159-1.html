2.1.5.9. Представление косинусоидальной и синусоидальной функций в виде комплексно-сопряженных чисел


Сумма комплексно-сопряженных чисел:

          C + = a + jb + a - jb = 2a = 2Re|C| = 2CcosYc
(3.24)

или C + = C(cosYc + jsinYc) + C(cosYc - jsinYc) = CejYc + Ce-jYc = 2CcosYc, откуда способ представления косинусоидальной функции с помощью комплексных величин cosYc = (ejYc + e-jYc)/2, т. е. косинусоидальная функция может быть представлена полусуммой двух комплексно-сопряжённых единичных векторов, имеющих начальные фазы +Yc и -Yc.

По аналогии, функция тока i(t) = Imcos(ωt + Yi) может быть представлена суммой двух комплексно-сопряжённых векторов, имеющих модуль Im/2 и вращающихся в противоположные стороны с одинаковой угловой частотой ω.
