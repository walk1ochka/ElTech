2.1.7.6. Представление синусоидальной функции в виде разности комплексно-сопряженных чисел


Разность комплексно-сопряженных чисел:

          C - = a + jb - a + jb = 2jb = 2Im|C| = 2jCsinYc

          или C - = CcosYc + jCsinYc - CcosYc + jCsinYc = CejYc - Ce-jYc = 2jCsinYc,

откуда:

          sinYc = (ejYc - e-jYc)/2j;

          CsinYc = C(ejYc - e-jYc)/2j при t = 0;

          Csin(ωt + Yc) = C[ej(ωt + Yc)- e-j(ωt + Yc)]/2j при t ≠ 0.

Если модуль комплекса С = Im , то

          Imsin(ωt + Yc) = Im [ej(ωt + Yc)- e-j(ωt + Yc)]/2j,

т. е. синусоидальная функция тока может быть представлена в виде разности двух комплексно-сопряжённых векторов, имеющих модуль Im/2 и вращающихся в противоположные стороны с одинаковой угловой частотой ω. Разность двух сопряжённых комплексов даёт мнимую величину, а потому её делят на j , чтобы получить действительную синусоидальную функцию.
