2.1.6.1. Три формы записи комплексного числа


Аналитически комплексное число С можно представить в алгебраической, тригонометрической и показательной формах (рис. 2.10):

          С = a + jb - алгебраическая,
          С = CejYc- показательная,
          С = C(cosYc + jsinYc) - тригонометрическая формы, т. е.

   С = CejYc = C (cosYc + jsinYc) = a + jb,
(2.15)

где С = |C|= - модуль комплексного числа С; Yc = arctg - аргумент комплексного числа С; a = Re[C] - действительная часть; b = Im[C] - мнимая часть комплексного числа С.
