1.2.6. Метод расчёта, основанный на законе Ома
Этот метод применяют для расчёта простых цепей с одним источником энергии. При этом преобразовывают пассивную часть схемы цепи для определения эквивалентного сопротивления относительно зажимов источника, затем входного тока и, посредством правила делителя тока, токов ветвей.
