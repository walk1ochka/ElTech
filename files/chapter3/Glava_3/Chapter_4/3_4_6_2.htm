4.4.6.2. Операторные схемы замещения и компонентные уравнения пассивных элементов

Выведенные соотношения между изображениями напряжений и токов пассивных элементов цепи и соответствующие им схемы замещения элементов во временной области и в области комплексного переменного р представлены в табл. 4.3. Каждому компонентному уравнению в изображениях соответствует операторная схема замещения элемента (см. четвертый столбец):

- резистору R (строка 1) - схема резистивного элемента с операторным сопротивлением ZR(p) = R (или с операторной проводимостью) YR(p) = 1/R;

- индуктивной катушке L (строка 2) без начального тока (iL(0-) = 0) - схема индуктивного элемента с операторным сопротивлением ZL(p) = Lp;

- конденсатору C (строка 4) без начального заряда (uC(0-) = 0) - схема ёмкостного элемента с операторным сопротивлением ZC(p) = 1/Cp;

- индуктивной катушке L c начальным током (строка 3) соответствуют две операторные схемы замещения, в которых начальный ток в катушке учитывается с помощью дополнительного источника: источника напряжения с ЭДС EL(p) = LiL(0-) , соединенного последовательно с индуктивным элементом (с сопротивлением ZL(p) = Lp) без начального тока, или источника тока с током JL(p) = iL(0-)/p, соединенного параллельно с индуктивным элементом (с проводимостью YL(p) = 1/Lp) без начального тока;

- конденсатору C с начальным зарядом q(0-) (строка 5) также соответствуют две операторные схемы замещения, в которых начальный заряд q(0-) или напряжение uC(0-) = q(0-)/C на зажимах конденсатора учитывается с помощью дополнительного источника: источника напряжения с ЭДС EC(p) = uC(0-)/p, соединенного последовательно с ёмкостным элементом (с сопротивлением ZC(p) = 1/Cp) без заряда, или источника тока с током JC(p) = CuC(0-), соединенного параллельно с ёмкостным элементом (с проводимостью YC(p) = Cp) без заряда.
