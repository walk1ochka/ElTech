Упражнение 3.4
Электрическая цепь (рис. 3.12) с элементами R, L и iL(0−) = 0 подключается к источнику синусоидального напряжения u(t = Umsin(ωt + Ψu). Определить ток iLL(t) по общей формуле (3.6).

Решение. 1. Обращаясь к (3.6), имеем
i(t) = iу(t∞) + [i(0+) − iу(0+)]e−t / τ, где τ = L / Rэ и Rэ = R.

где Um = Ume jΨu; Z = Ze j arctg(XL / R) = Zejφ.
3. Мгновенное значение установившегося после коммутации тока
iу(t∞) = Im sin(ωt + Ψu − φ),    iу(0+) = Iуsin(Ψu − φ).
4. Согласно 1ПК и условию задачи ток iL(0+) = iL(0−) = 0. Следовательно, ток в переходном процессе
i(t) = Imsin(ωt + Ψu − φ) − Imsin(Ψu − φ)e−t / τ.
5. Характер переходного процесса в цепи зависит от момента включе­ния цепи:

• если при t = 0+ начальная фаза тока Ψi = 0 (sin(Ψu − φ) = 0), то сразу же наступает установи­вшийся режим. Если включение происходит при фазе Ψi = ±π/2, то ток i(0+) = 0, так как его свободная и установившаяся составляющие, равные амплитуде Im тока iу, имеют разные знаки (рис. 3.13а). При t = T/2 ток достигает максимального значения, не превышающего удвоенной амплитуды Im установившегося тока;
• если включение цепи происходит при фазе тока 0 ≠ Ψi ≠ ±π/2, то в возникшем переходном процессе будет меньшее увеличение амплитуды тока i(t) (рис. 3.13б), чем при включении цепи при фазе Ψi = ±π/2.
