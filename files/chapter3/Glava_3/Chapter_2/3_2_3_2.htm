4.2.3.2. Определение постоянной времени цепи

 

Постоянную времени цепи с одним реактивным элементом определяют следующим образом:

  а) для цепи с индуктивным элементом L: t = L/RЭ;
б) для цепи с ёмкостным элементом C: t = RЭC ,

где RЭ - входное сопротивление цепи относительно зажимов удалённого из схемы реактивного элемента, в которой идеальные источники напряжения замкнуты накоротко, а ветви с источниками тока разомкнуты.

Примечание. 1. В цепях постоянного тока (u(t) = U) послекоммутационное значение функции fy(0+) = fy(t). Например, напряжение на конденсаторе uСy(0+) = uСy(t) и т. д.

2. В цепях переменного тока численное значение установившейся составляющей переходной функции fy(0+) находят из выражения fy(t) путём подставки t = 0. Например, напряжение на конденсаторе
     uCy(0+) = UmСcos(wt + Yu - j)|t=0 = UmСcos(Yu - j).
