Как отмечалось, полное значение любой переходной величины (тока, напряжения) равно сумме установившейся составляющей fу(t∞), обусловленной входным воздействием u(t), и свободной составляющей fсв(t) = Ae−t /τ, обусловленной накопленной энергией в цепи, т.е.
f(t) = fу(t∞) + fсв(t) = fy(t∞) + Ae−t/τ.
При t = 0+ переходная функция f(0+) = fy(0+) + A. Отсюда постоянная интегрирования, зависящая от параметров цепи, входного воздействия u (t) и момента включения, A = f(0+) − fy(0+).
Итак, общая формула решения дифференциального уравнения для любой линейной цепи с одним накопителем энергии с постоянной или периодической действующей в цепи функцией u(t) имеет вид
f(t) = fy(t∞) + [f(0+) − fy(0+)]e−t/τ.(3.6)
За интервал времени t = τ функция Ae−t/τ = [ƒ(0+) − ƒ (0+)]e−t/τ уменьшается в е ≈ 2,72 раза.
Постоянную времени цепи с одним реактивным элементом определяют следующим образом:
• для цепи с индуктивным элементом L: τ = L/Rэ,
• для цепи с ёмкостным элементом C: τ = RэC,
где Rэ − входное сопротивление цепи относительно зажимов удалённого из схемы реактивного элемента, в которой идеальные источники напряжения замкнуты накоротко, а ветви с источниками тока разомкнуты.
