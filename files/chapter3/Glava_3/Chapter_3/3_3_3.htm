3.3.3. Формирование уравнений состояния методом наложения
Заменив индуктивные и ёмкостные элементы в послекоммутационной схеме соответственно источниками тока и напряжения, получим резистивную схему с дополнительными источниками.
Записав напряжение uLk = L[diLk/dt] и ток iCk = C[duCk/dt] для каждого индуктивного и ёмкостного элемента, составляют систему уравнений состояния для п схем, в каждой из которых оставлен один источник энергии. Полученную методом наложения систему уравнений состояния цепи решают, как правило, посредством численных методов на ЭВМ.
Для конкретизации этапов формирования уравнений состояния методом наложения рассмотрим послекоммутационную схему (см. рис. 3.26б), выделив в ней ёмкостный С и индуктивный L элементы (рис. 3.27а) таким образом, чтобы оставшаяся часть схемы содержала только резистивные элементы R1 и R2 и источник напряжения е.

Согласно принципу компенсации индуктивный элемент L заменим источником тока JL = iL, который доставляет ток в том же направлении, что и в исходной схеме, а ёмкостный элемент С − источником напряжения с ЭДС EC = uC, направленной встречно току iC (рис. 3.27б).
